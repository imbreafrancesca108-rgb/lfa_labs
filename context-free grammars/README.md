# Context-Free Grammar Emulator
## Description 
This project implements a Context-Free Grammar emulator in Python. 
The grammar configuration is loaded from external configuration files, while the program verifies whether a given word can be generated by the grammar or not.
In this example, the grammar recognizes all strings having a subsequence of 0s followed by a subsequence of 1s, both having the same length. 
## Project Structure
```
context-free grammars/
│
├── config.cfg
├── grammar.cfg
├── emulator.py
└── executabil.py
```
### config.cfg 
Contains the name of the grammar configuration file that will be loaded by the program. 
### grammar.cfg 
Contains the grammar definition:
- variabiles
- terminals
- start symbol
- production rules
- 
### emulator.py 
Contains 4 functions which : 
- read the configuration files
- parse grammar sections
- build the grammar
- verify input words from the user
### executabil.py 
Main program file, which: 
- reads the configuration file
- loads the grammar
- receives input words from the user
- displays a message on whether the word is accepted or rejected
## Recognized Language 
L = {0^n1^n | n>=0 } 
## How the Grammar Works 
S-> 0S1 adds one 0 to the left and one 1 to the right

S-> e stops the derivation

Example of derivation:
S
=> 0S1
=> 00S11
=> 000S111
=> 000e111
=> 000111

## Running the Program
```
Run:
python executabil.py
Type your word: 01
ACCEPTED
Type your word: 00001111

Type your word: 010
REJECTED
Type your word: Exit
The command `Exit` terminates the program.
```
